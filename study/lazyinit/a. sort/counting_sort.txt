# 설명 (알고리즘)
1. 정렬할 배열의 값이 몇번 등장하는지 카운팅 배열에 저장
2. 카운팅 배열을 누적 배열로 변경.
3. 정렬할 배열의 값을 인덱스로하여 누적 배열의 값을 받아오고 그 값을 인덱스로 하여 새로운 배열에 배열의 값을 저장
4. 누적배열의 값을 1 감소
5. 3.~4.를 n번 반복

# point
- Suffix Array 생성 등 문자열에 관련한 정렬일 경우 효과적

# 안정성
 - True

# 정렬기법
 - 내부정렬

# 복잡도

1. 시간복잡도 :
    - best : O(n)
    - avg : O(n)
    - worst: O(n)

2. 공간복잡도 : O(n + max_num of n)
